# Sync GitLab La Poste (usage interne)

Scripts one-way pour synchroniser le squelette plateforme vers un dépôt GitLab La Poste.

- `sync-to-laposte.sh` — rsync + commit (ne pas éditer le clone La Poste à la main)
- Configurer `GITLAB_LAPOSTE_REMOTE` dans `.env` (non commité)

Voir aussi la doc déploiement interne dans ce dossier si fournie par votre organisation.
